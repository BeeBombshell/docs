Whether you're a new user just getting started with the platform or an experienced developer looking to solve a specific issue, this page is designed to provide you with the information and resources you need to make the most of Rowy.

## Getting Started

If you're new to Rowy, we recommend that you start with the following resources:

- [Rowy Documentation](https://docs.rowy.io/): We constantly try to improve our documentation to make it more comprehensive. Rowy Docs covers **everything from installation to usage and advanced features**. The documentation also includes code samples, tutorials, and frequently asked questions to help you get started.

- [Rowy GitHub Repository](https://github.com/rowyio/rowy): The official Rowy repository on GitHub, where you can find the **latest release, report issues, and contribute to the project**. The repository will help you stay up-to-date with the latest developments.

- [Rowy Community Discussions](https://github.com/rowyio/rowy/discussions): The Rowy Community is a place where you can **ask questions, share ideas, and discuss best practices**. You can browse existing discussions or start a new one to get help from the community and thr Rowy Team.

- [Rowy Discord Server](https://discord.com/invite/B8yAD5PDX4): Our Discord server is a great place to chat with other Rowy users and get help from the Rowy team. You can join the server by following the link above.

Our goal is to ensure that you have everything you need to succeed with Rowy, whether you're building a simple application or a complex enterprise solution. So take a look around, and don't hesitate to reach out if you have any questions or issues. We're here to help!
