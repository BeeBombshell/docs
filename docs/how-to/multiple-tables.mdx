Multiple tables on the same collection can display different subsets of data for
different users.

## Video Demo
<Video controls url="https://youtu.be/4Y60tNBdLv4" />

## Steps

1. Click the **+** button in the bottom-right corner of the Home page.
2. Choose which **Collection** to display.
3. Set the **Table name** and other display options.
4. Set a unique **Table ID**.
5. Set who can view and edit the table in **Access controls**. Make sure you
   [set the corresponding roles for your users](../how-to/roles.mdx).
6. Optionally enable **Auditing** for the table.
7. Optionally, initialize your table with columns.

## Notes

- If you delete a table, only the Rowy table config will be removed. No
  collection data will not be deleted.
